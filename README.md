# *SimpleGalleryViewer*

## Install
#### Debian
`sudo apt install pkg-config libcairo2-dev gcc python3-dev python3-pip libgirepository1.0-dev -y`
`python3 -m pip install --user -U PyGobject`
#### ArchLinux
`sudo pacman -Syu python-pip gtk3 gobject-introspection`
`python3 -m pip install --user -U PyGobject`

## Running
`python3 sgv.py`
